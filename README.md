## SpringBoot-LayIM-Scala ###

### 主要技术 ### 

    LayIM
    WebSocket
    Gradle
    Spring Boot
    Scala
    Redis
    Alibaba Druid
    Java Mail
    Mybatis And PageHelper
    Spring Security

### 环境 ###

* Scala 2.11.8
* JDK 1.8
* Gradle 3.0
* Mysql
* Redis 

### 使用 ###

配置Mysql数据库，Redis以及邮件服务器，如果不需要邮件相关服务，可以在UserService.scala中注释掉相关的代码

启动：Application.scala

访问：http://localhost

缺少接口：

1. 查询我创建的群接口
2. 底部查询接口
3. 创建群组接口
4. 管理群组接口
5. 管理好友列表接口
6. 更新个人信息接口
